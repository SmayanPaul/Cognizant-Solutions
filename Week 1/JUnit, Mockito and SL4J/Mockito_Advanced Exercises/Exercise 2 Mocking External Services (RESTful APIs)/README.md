# Exercise 2: Mocking External Services (RESTful APIs)

## Description
This exercise covers mock REST API client dependencies in services using Mockito to verify basic service layer logic.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\advanced\ApiServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.advanced.ApiServiceTest
```
