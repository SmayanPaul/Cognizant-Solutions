# Exercise 5: Mocking Multiple Return Values

## Description
This exercise covers mock dependencies returning different values on consecutive method calls using Mockito to verify stateful business logic.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\advanced\MultiReturnServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.advanced.MultiReturnServiceTest
```
