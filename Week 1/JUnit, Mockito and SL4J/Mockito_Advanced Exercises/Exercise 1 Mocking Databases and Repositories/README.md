# Exercise 1: Mocking Databases and Repositories

## Description
This exercise covers mock repository dependencies in services using Mockito to verify basic business logic.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\advanced\ServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.advanced.ServiceTest
```
