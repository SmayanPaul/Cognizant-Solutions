# Exercise 3: Mocking File I/O

## Description
This exercise covers mock file reader and writer dependencies using Mockito to verify and test business logic that manipulates filesystem files.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\advanced\FileServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.advanced.FileServiceTest
```
