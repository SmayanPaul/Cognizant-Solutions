# Exercise 4: Mocking Network Interactions

## Description
This exercise covers mock network socket or client dependencies using Mockito to verify and test business logic that triggers external network connections.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\advanced\NetworkServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.advanced.NetworkServiceTest
```
