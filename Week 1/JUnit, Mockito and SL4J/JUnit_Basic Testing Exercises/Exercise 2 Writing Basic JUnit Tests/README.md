# Exercise 2: Writing Basic JUnit Tests

## Description
This exercise covers writing basic JUnit tests for a simple Java class (`Calculator`) containing basic arithmetic methods (`add` and `subtract`).

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\junit\basic\CalculatorTest.java
```

### 2. Run
Run the tests using the JUnit console runner:
```cmd
java -cp "bin;..\lib\*" org.junit.runner.JUnitCore com.junit.basic.CalculatorTest
```
