# Exercise 3: Assertions in JUnit

## Description
This exercise covers utilizing various JUnit 4 assertions (`assertEquals`, `assertTrue`, `assertFalse`, `assertNull`, `assertNotNull`, `assertArrayEquals`, `assertSame`, `assertNotSame`) to validate distinct test conditions.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\junit\assertions\AssertionsTest.java
```

### 2. Run
Run the tests using the JUnit console runner:
```cmd
java -cp "bin;..\lib\*" org.junit.runner.JUnitCore com.junit.assertions.AssertionsTest
```
