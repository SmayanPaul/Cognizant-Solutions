# Exercise 6: Verifying Interaction Order

## Description
This exercise covers verifying the order in which mock methods are called using Mockito's `inOrder()` verification.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\exercises\MyServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.exercises.MyServiceTest
```
