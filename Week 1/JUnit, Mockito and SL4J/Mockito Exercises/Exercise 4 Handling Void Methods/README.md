# Exercise 4: Handling Void Methods

## Description
This exercise covers stubbing and verifying interactions with void methods using Mockito's `doNothing()` and `verify()` constructs.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\exercises\MyServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.exercises.MyServiceTest
```
