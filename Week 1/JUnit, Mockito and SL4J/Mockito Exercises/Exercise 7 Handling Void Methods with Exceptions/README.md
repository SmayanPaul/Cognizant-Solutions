# Exercise 7: Handling Void Methods with Exceptions

## Description
This exercise covers stubbing a void method to throw an exception using Mockito's `doThrow()` construct, verifying exception handling/propagation, and verifying the mock interaction.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\exercises\MyServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.exercises.MyServiceTest
```
