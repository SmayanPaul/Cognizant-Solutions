# Exercise 2: Verifying Interactions

## Description
This exercise covers creating a mock object, calling its methods with specific arguments, and verifying the interaction using Mockito's `verify()` method.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\exercises\MyServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.exercises.MyServiceTest
```
