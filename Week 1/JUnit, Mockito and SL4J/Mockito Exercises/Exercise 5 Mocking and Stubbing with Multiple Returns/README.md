# Exercise 5: Mocking and Stubbing with Multiple Returns

## Description
This exercise covers stubbing methods to return different values on consecutive invocations using Mockito's chained `thenReturn()` builder.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\mockito\exercises\MyServiceTest.java
```

### 2. Run
Run the test program:
```cmd
java -cp "bin;..\lib\*" com.mockito.exercises.MyServiceTest
```
