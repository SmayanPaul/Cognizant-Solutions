# Exercise 4: Exception Testing

## Description
This exercise covers validating that methods throw expected exceptions under failure conditions in JUnit 5 using the `assertThrows` assertion.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\junit\advanced\exception\ExceptionThrowerTest.java
```

### 2. Run
Run the tests using the programmatic launcher in the class:
```cmd
java -cp "bin;..\lib\*" com.junit.advanced.exception.ExceptionThrowerTest
```
