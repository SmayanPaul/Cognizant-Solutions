# Exercise 1: Parameterized Tests

## Description
This exercise covers creating a class `EvenChecker` with a method `isEven(int number)` and validating it using JUnit 5's `@ParameterizedTest` and `@ValueSource` annotations to test multiple input sets.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\junit\advanced\parameterized\EvenCheckerTest.java
```

### 2. Run
Run the test program using the launcher in the class:
```cmd
java -cp "bin;..\lib\*" com.junit.advanced.parameterized.EvenCheckerTest
```
