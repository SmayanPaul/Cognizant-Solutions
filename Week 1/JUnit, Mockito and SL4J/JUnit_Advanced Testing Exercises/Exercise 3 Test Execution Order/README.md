# Exercise 3: Test Execution Order

## Description
This exercise covers controlling the sequence of test case execution in JUnit 5 using `@TestMethodOrder` and `@Order` annotations.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\junit\advanced\order\OrderedTests.java
```

### 2. Run
Run the tests using the programmatic launcher in the class:
```cmd
java -cp "bin;..\lib\*" com.junit.advanced.order.OrderedTests
```
