# Exercise 5: Timeout and Performance Testing

## Description
This exercise covers performance testing in JUnit 5 to assert that execution completes within a specified time limit using the `assertTimeoutPreemptively` assertion.

## How to Compile & Run

### 1. Compile
Run the following command from this directory:
```cmd
javac -cp "..\lib\*" -d bin src\com\junit\advanced\timeout\PerformanceTesterTest.java
```

### 2. Run
Run the tests using the programmatic launcher in the class:
```cmd
java -cp "bin;..\lib\*" com.junit.advanced.timeout.PerformanceTesterTest
```
