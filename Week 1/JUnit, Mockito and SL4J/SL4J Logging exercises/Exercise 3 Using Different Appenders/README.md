# Exercise 3: Using Different Appenders

## Description
This exercise demonstrates configuring Logback Console and File Appenders in a `logback.xml` file, which records logged messages both on stdout and in an external `app.log` file.

## How to Compile & Run

### Run Application
From this directory, compile and run the application:
```cmd
d:\Study\Cognizant\apache-maven-3.9.6\bin\mvn.cmd compile exec:java
```
